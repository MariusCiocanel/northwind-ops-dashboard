# Welcome 👋

This folder is your starter kit for **Building productivity superpowers with Claude Code** — a Generalist World Summer School workshop. Event page: https://luma.com/i0dkufzc

You don't need any coding experience. Claude Code does the building; you do the directing.

## The scenario

You've just joined the ops team at **Northwind Coffee Co.**, a direct-to-consumer coffee roaster. Leadership keeps the numbers in a couple of spreadsheets and reviews them in a weekly sync. Your job: turn that into a real dashboard, a briefing the team can read each morning, and an automation that keeps it fresh — all with Claude Code.

## What's in here

```
.
├── CLAUDE.md     ← project context Claude reads automatically
├── README.md     ← you are here
├── context/      ← who the team is + what matters this week (read first)
└── data/         ← the raw spreadsheets (sales + inventory)
```

## How to start

1. Open this folder in Claude Code.
2. Ask Claude to read the files in `context/` and take a look at the data, then tell you what it understands about the project.
3. Follow along with the workshop from there.

Everything Claude needs is already here. Have fun.
